Κίνδυνοι και κανόνες ασφαλείας στο διαδίκτυο

Συχνά μας ρωτάτε πώς μπορείτε να προστάτεψε το παιδί σας κι εσάς τους ίδιους, από τους κινδύνους του διαδικτύου. Τι θα μπορούσατε να εγκαταστήσετε στον υπολογιστή ή το κινητό ώστε να αποφύγετε τους κινδύνους στο διαδίκτυο ή πώς θα παρακολουθείτε τις κινήσεις που κάνει το παιδί σας σε αυτό.

Συνήθως η απάντηση μας είναι ότι εκτός από ένα πρόγραμμα για την αντιμετώπιση της απειλής από τους ιούς δεν υπάρχει κάτι άλλο παρά μόνο να εισαχθούν κάποιοι κανόνες ασφαλείας. Κρίνεται αναγκαίο να γίνει συζήτηση μεταξύ εσάς και του παιδιού όσον αφορά τους προαναφερθέντες κινδύνους αλλά και να ενημερωθείτε για κάθε νέο τρόπο παραπλάνησης με τον οποίο προσπαθούν οι επιτήδειοι να κερδίσουν κάτι από εσάς ή το παιδί σας.

Το πρώτο πράγμα που θα πρέπει να συνειδητοποιήσει κάθε χρήστης του διαδικτύου είναι ότι ο παγκόσμιος ιστός ( INTERNET ) δεν διαφέρει σε τίποτα από μια πραγματική κοινωνία. Κρύβει λοιπόν, αντίστοιχους κινδύνους και απαιτεί αντίστοιχα μέτρα προστασίας από αυτούς. Η σωστή τακτική ενημέρωση σχετικά με την φύση των κίνδυνων αλλά και τον τρόπο αντιμετώπισης τους γίνεται απαραίτητη προκειμένου να ελαχιστοποιηθεί ο κίνδυνος έκθεσης του χρήστη στους βασικότερους τουλάχιστον κινδύνους.

  • Κίνδυνοι στο διαδίκτυο

    1. Αποπλάνηση Ανηλίκων ( Grooming )

      2. Προσπαθεί να πείσει το παιδί να του στείλει προσωπικές φωτογραφίες ή video, ή κι ακόμα να του πει πού πηγαίνει σχολείο ή σε ποια περιοχή μένει, ώστε να καταφέρει μέσω όλων αυτών να συναντηθεί μαζί του.

    2. Ηλεκτρονική Παρενόχληση ( Cyberbulling )

      3. Η συνεχής επιθετική συμπεριφορά εναντίον κάποιου μέσω διαδικτύου έτσι ώστε να αποκτηθεί δύναμη και εξουσία πάνω του ( ψηφιακός εκφοβισμός )

    3. Παραπλανητικά Μηνύματα ( Scam )

      4. Πρόκειται για παραπλανητικά μηνύματα το οποία σκοπό έχουν να δελεάσουν τους παραλήπτες με την προσφορά μεγάλων χρηματικών ποσών, δώρων (ταξίδι, αυτοκίνητο, μηχανή κ.α.), ή ευκαιριών καριέρας.

    4. Ηλεκτρονικό Ψάρεμα ( Phishing, Pharming )

      5. Phishing είναι η προσπάθεια υποκλοπής ευαίσθητων προσωπικών δεδομένων, όπως κωδικών, αριθμών πιστωτικών καρτών, αριθμών λογαριασμών κι άλλων προσωπικών στοιχείων με ψεύτικα προσχήματα.

    5. Ιοί ( Virues, Trojan )

      6. Μικρά λογισμικά που μεταφέρονται και τρέχουν στον υπολογιστή με στόχο να αλλάξουν ή να καταστρέψουν, χωρίς ο χρήστης να το γνωρίζει. Τα συνήθη προβλήματα που προκαλούν είναι η διαγραφή αρχείων, αποστολή ηλεκτρονικών μηνυμάτων, δημιουργία εικονικής κίνησης προς το διαδίκτυο, απασχόληση πόρων του υπολογιστή. Η διάδοση τους γίνεται κυρίως από το ηλεκτρονικό ταχυδρομείο (email) διαφόρων δωρεάν προγραμμάτων κι από κάποιες ιστοσελίδες.
